üîå API Integration

The app integrates with Koneksi Storage API using these endpoints:

List Videos:
GET /api/clients/v1/directories/{directory_id}
Upload Video:
POST /api/clients/v1/files?directory_id={directory_id}
Stream Video:
GET /api/clients/v1/files/{file_id}/download
Delete Video:
DELETE /api/clients/v1/files/{file_id}
